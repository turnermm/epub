Sie können dieses Buch neu-erstellen, indem Sie die Seiten bearbeiten, die Seitenliste ändern, oder die Reihenfolge der Seiten ändern, etc.  
Wenn Sie das Buch fertiggestellt haben, sollten Sie seine ID von der Liste der zu-bearbeitenden Bücher entfernen.   
Das können Sie mit einem Klick auf den Button "Seite von Bearbeitungs-Liste entfernen" machen. Dadurch wird die ID für andere Autoren freigegeben. 
Bis Sie das machen, werden auf der Seite weiterhin die Buttons **Start** und **Seite entfernen...** erscheinen, 
selbst wenn Sie die epub-Syntax von der Seite entfernt haben.
